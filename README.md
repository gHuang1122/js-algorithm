# js-algorithm
基于js实现常规加密算法

## 位运算符
- & (按位与)
```
同为1,值为1
   0011 1100
 & 0000 1111
 = 0000 1100
作用：用于取出指定位数的二进制
1.例如： 想把后四位4位取出来
        0011 1001
      & 0000 1111 (0xF)
      = 0000 1001
2.例如：使用位移取出前4位
        1011 1001 dec(185)
      >>        4 dec(4)
      = 0000 1011 dec(11)
```
- | (按位或)
```
 有一位是1,就是1
```
- ^ (按位异或)
```
不同为1,相同为0
    0100 0101
  ^ 0000 0001
  = 0100 0100
  ^ 0000 0001 (异或同一个数)
  = 0100 0101 (还原了)  
可以用于加密,例如：
    A ^ B = C, A是明文，B是密钥，C是密文。C ^ B = A 即可还原
```
- 结合使用
```
// 可以运用到权限上

假设对于一个文件有CRUD的权限
    C(create): 0000 0001
    R(read):   0000 0010
    U(update): 0000 0100
    D(delete): 0000 1000
    .....
    增加新权限在新的位置上赋值1即可
通过 | 创造新的权限：
    CR: C | R = 0000 0011
    CD: C | D = 0000 1001
    CRD: CR | CD = 0000 1011
    CRUD: CRD | U = 0000 1111

角色1：CR 权限
角色2：CRD 权限
角色3：CRUD 权限

如何判断一个角色是否具备某个权限：
    1. 判断 角色2 是否具备 U 权限
       0000 1011 & 0000 0100 = 0000 0000 不具备
    2. 判断 角色2 是否具备 C 权限
       0000 1011 & 0000 0001 = 0000 0001 具备
    3. 判断 角色2 是否具备 CR 权限
       0000 1011 & 0000 0011 = 0000 0011 具备
    即：角色权限 & 权限 == 权限 (权限大于0)
```
